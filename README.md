# 🎨 Computer Graphics Projects Gallery

A curated collection of real-time interactive graphics systems built using **JavaScript** and the **HTML5 Canvas API**.

This repository explores animation systems, physics-inspired simulation, parametric motion, and interactive rendering techniques.

---

## 🌌 Projects

### 🔥 Firework Physics Simulation
A real-time firework system simulating projectile motion under gravity and air resistance.

**Key Features**
- Analytical launch velocity computation for guaranteed target convergence
- Gravity-based motion with frame-rate normalization
- Reverse-integrated motion trails for curved trajectory reconstruction
- Procedural particle explosion system (50–250 particles per burst)
- Alpha decay and stochastic luminous burn effects
- Interactive slider to control firework frequency

**Core Concepts**
- Kinematics under constant acceleration  
- Delta-time scaling  
- Procedural particle systems  
- Alpha blending and layered rendering  

---

### 🚁 Quadcopter Flight Simulator
A parametric drone simulation featuring autonomous motion behaviors and interactive controls.

**Key Features**
- Parametric flight paths (circular, figure-eight, banked weave)
- Tangent-aligned heading via trigonometric modeling
- Frame-rate independent animation loop
- WASD-controlled drone navigation
- Projectile system with lifetime management
- Layered lighting effects using radial gradients

**Core Concepts**
- Parametric equations  
- Transformation hierarchy (translate / rotate / scale)  
- Time-based motion modeling  
- Interactive input state management
